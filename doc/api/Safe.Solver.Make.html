val solve_constraints : ?optimize_join:bool ->
       'a ->
       Constraints.cst list ->
       Constraints.State.t -> Constraints.State.Var.t list -> Constraints.State.t
solve_constraints ~optimize_join prog csts state init computes the fixpoint of the constraints csts, starting from the initial state state by applying the constraints that depends on nothing or on initial variables init. If optimize_join is true, then it tries to avoid joining useless values, at the cost of some additional computations.
