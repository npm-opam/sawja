val get_first_pp : 'a JProgram.program ->
       Javalib_pack.JBasics.class_name ->
       Javalib_pack.JBasics.method_signature -> 'a t
get_first_pp p cn ms gets a pointer to the first instruction of the method ms of the class cn.
Raises
  • Not_found if cn is not a class of p, or ms is not a method of cn.
  • NoCode if the method ms has no associated code.
val get_first_pp_wp : 'a JProgram.node ->
       Javalib_pack.JBasics.method_signature -> 'a t
get_first_pp_wp n ms gets a pointer to the first instruction of the method ms of the node n.
Raises NoCode if the method ms has no associated code.
val goto_absolute : 'a t -> int -> 'a t
goto_absolute pp i: return the program pointer at instruction i in the method of program pointer pp.
val goto_relative : 'a t -> int -> 'a t
goto_relative pp i: return the program pointer at instruction i'+i (with i' the current program point of pp)in the method of program pointer pp.

val static_pp_lookup : 'a JProgram.program -> 'a t -> 'a t list
static_pp_lookup program pp: Return a list of method entries which can be reached by a call from the current program pointer. It returns fully implemented methods pp only. The computation is based on the RTA or CRA result (depending on the one used to build the program).
