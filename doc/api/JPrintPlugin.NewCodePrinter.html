Precise warning on a program point: LineWarning of JPrintPlugin.warning_pp is automatically generated from the data structure for warnings from JPrintPlugin.plugin_info. This warning could be transformed in a PreciseLineWarning in the PrintInterface.to_plugin_warning function provided for the specific code representation.
