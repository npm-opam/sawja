Documentation

High level manipulations of programs

JProgram
Defines high level OCaml representation of a Java bytecode program.
JControlFlow
Provides some JVM resolution and lookup functions and allows some navigation in the control flow graph of a program for static analysis.
JType
It provides some type utilities (mainly subtyping).
JNativeStubs
Allows to manipulate abstract data structures representing native methods object allocation and method calls.
JCRA
Builds high level representations of Java bytecode programs using basic Class Reachability Analysis.
JRTA
Builds high level representations of Java bytecode programs using Rapid Type Analysis algorithm.
JRRTA
Builds high level representations of Java bytecode programs using a refinement of RTA.
XTA
Refine the call graph of a program using the XTA class analysis.

Intermediate Representations

JBir
Stackless, unstructured intermediate representation for Java Bytecode, in which expression trees are reconstructed and method and constructor calls are folded.
A3Bir
Stackless, 3-address like and unstructured intermediate representation for Java Bytecode, in which basic expression trees are reconstructed and method and constructor calls are folded.
JBirSSA
SSA form of the JBir intermediate representation.
A3BirSSA
SSA form of the A3Bir intermediate representation.

Tools for static analysis
Fixpoint solvers

Iter
Defines a simple fixpoint solver for constraints between integer values (suitable for constraints between program points).
Safe
Defines a fixpoint solver managing domains for differents levels of the code representation (global, class, field, method, program point).

Examples of static analyses

Live_bir
Live variable analysis for JBir representation (use Iter solver).
ReachDef
Reachable definitions analysis for JBir representation (use Iter solver).
AvailableExpr
Available expressions analysis for JBir representation (use Iter solver).
ReachableMethods
ReachabeMethods allows to compute the methods of a program that are reachable from a set of entry points (use Safe solver).

Printing programs

JPrintHtml
Pretty-Html-printer for high level programs.

Primitives for generating warnings and information for the Sawja Eclipse Plugin
